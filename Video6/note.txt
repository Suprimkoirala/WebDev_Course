Core Web Vitals
1. CLS: Cumulative Layout Shift
CLS is a measure of the largest burst of layout shift scores for every unexpected layout shift that occurs during the entire lifespan of a page.A layout shift occurs any time a visible element changes its position from one rendered frame to the next. There should be minimum layout shift in the web page to improve the user experience and rank higher.
2. LCP - Largest Contextual Paint
75th percent of page load should be within 2.5 second to improve rank score. It defines how long it takes to load the largest element of the page.
3. FID-First Input Delay
How much delay is between the first input given by the user and output. Good FID score is 100ms or less.
